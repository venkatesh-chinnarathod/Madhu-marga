# Madhu-Marga: The Digital Beekeeper's Diary 🍯

**Madhu-Marga** is a data-driven Android application designed to modernize apiculture (beekeeping) in India. Developed as part of the **MindMatrix VTU Internship**, it serves as an AI-guided assistant to help farmers monitor hive health, track honey production, and prevent colony loss.

## 🌟 Project Vision: The "Sweet Revolution"
Beekeeping is vital for biodiversity and provides a sustainable secondary income for farmers. **Madhu-Marga** addresses the sensitivity of bee colonies to pests and environmental factors by providing a "Digital Diary" that offers real-time intervention logic.

## ✨ Key Features

### 1. AI-Guided Intervention Alerts 🚨
Uses a **Decision Matrix** to analyze hive observations.
* **Scenario:** If a user logs "Low Activity" in the inspection checklist...
* **Result:** The app triggers an **Intervention Alert** advising the farmer to check for queen health or starvation, preventing colony absconding.

### 2. Digital Inspection Log 📝
Replaces paper logs with cloud-synced checklists tracking:
* Queen presence and egg-laying patterns.
* Pest detection (Varroa mites, Wax moths).
* Hive temperament and strength.

### 3. Harvest Tracker & Analytics 📊
* **Year-over-Year Comparison:** Visualizes honey yield growth across different seasons.
* **Honey Flow Progress Bar:** A visual indicator of the current nectar-gathering season.

### 4. Flora Calendar 🌸
A localized guide mapping blooming cycles of Indian flora (Mustard, Sunflower, Neem) to help beekeepers plan hive movements and optimize pollination.

## 🛠️ Technical Stack

* **UI Framework:** Jetpack Compose (Modern, responsive UI)
* **Backend:** Firebase Firestore (Real-time data synchronization)
* **Design:** Professional "Honey/Yellow" theme aesthetic
